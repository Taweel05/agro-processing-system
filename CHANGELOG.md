# Журнал изменений (CHANGELOG) для Агро-системы

## [1.0.0] – 2026-05-24 (дата завершения второй практики)

### Добавлено
- Регистрация и авторизация пользователей с тремя ролями (клиент, рабочий, администратор).
- Модель GrainBatch для учёта партий зерна.
- Форма добавления партии с русскоязычными полями.
- Интеграция с внешним API погоды (OpenWeatherMap) – при создании партии автоматически подставляются температура и влажность.
- Страница списка партий с отображением погоды.
- Базовые тесты (создание пользователя, создание партии, работа API).
- Конфигурация Docker для контейнеризации (Dockerfile, docker-compose.yml).

### Изменено
- Кастомная модель User добавлено поле role.
- Шаблоны переведены на русский язык.

### Исправлено
- Ошибка импорта `requests` (установлен пакет).
- Синтаксическая ошибка в views.py (убрал лишний импорт между декораторами).
- Отсутствие поля weather_info в модели (добавлено миграцией).

### Безопасность
- Отключён DEBUG в production-настройках (в .env).
- Секретный ключ вынесен в переменные окружения.
